# spacing mixin

Mixins are functions that accept a space enum + the theme and return styles. The spacing function is fairly simple and helps to ensure consistency in UI.
The spacing value can take different shapes to make it easy to apply any combination of spacings.

## Usage examples

Spacing function accepts space string or objects, returns margin and apply spacing.
Below are examples of using the spacing() function. Pass to the function the value you wish to use.

### Apply individual spacing

You can set the individual spacing to one side when passing one value in an object (example, top: 'giga'). You can use the following values to define the spacing:

- 'bit',
- 'byte',
- 'kilo',
- 'mega',
- 'giga',
- 'tera',
- 'peta',
- 'exa',
- 'zetta',
