## Usage in code

```javascript
import styled, { css } from 'react-emotion';

const Box = styled('div')`
  ${({ theme }) => css`
    background-color: ${theme.colors.n100};
    color: ${theme.colors.success};
  `};
`;
```

## Best practices for using color

* Don't refer to hex values in your code (e.g. `#DAEAFF`) but rather use the
  colors via the theme (e.g. `theme.colors.p100).
* When referring to a color that should be treated as _primary_ (such as a
  button or a link), be sure to use the p-family of colors rather than
  the b-family of colors.
