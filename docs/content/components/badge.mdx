## Best practices
- Only use one badge at a time.

## Usage guidelines

- Use to indicate statuses of a process, for instance, "Pending payment" or
  "Successful transaction".
- Only use semantic badges in cases where their colors can be mapped to a
  meaning.

## Component variations

### Semantic badges
The badges come in a number of colors based on their semantic meaning.

Badges that receive the `circle` prop can be used to indicate notifications.
