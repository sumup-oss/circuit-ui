## Variations

If you need more control on the tabs, you can build you own tabs list instead of passing an array of items to the `Tabs` component. This can be useful to style any of the components individually.

<Playground>
  <TabList>
    <Tab selected>Tab 1</Tab>
    <Tab>Tab 2</Tab>
    <Tab>Tab 3</Tab>
  </TabList>
  <TabPanel style={{ padding: '12px' }}>
    Bear in mind that you'll have to manually implement the interface for
    changing contents and the selected indicator.
  </TabPanel>
</Playground>
