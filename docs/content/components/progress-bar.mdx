# Progress bar

<Status.Stable />

A progress bar is a UI element used to visualise the progression of a task or
operation. The visual element is always combined with a label describing a
status value either in percentage or fraction format.

<Playground>
  <ProgressBar max={100} value={30} size={ProgressBar.KILO}>
    30%
  </ProgressBar>
</Playground>

<Props of={ProgressBar} />

## When to use it

The progress bar should be used to indicate the amount of time left to complete
a task. It should not however, be used to indicate the time needed to load a page.

## Usage guidelines

- **Do** place the status value to the right of the progress bar.
- **Do** use in moderation, only for tasks, operations or information that may take a long time.
- **Do not** use an indeterminate value beyond percentage or fraction.
- **Do not** use a progress bar for every task, operation or information.

## Component variations

### Percentage or fraction completion

When you specify the label for the progress bar, it should only be shown
as either a fraction (e.g. 1/3) or percentage (e.g. 33%).

<Playground>
  <Fragment>
    <ProgressBar max={3} value={1} size={ProgressBar.KILO}>
      1/3
    </ProgressBar>
    <ProgressBar max={3} value={1} size={ProgressBar.KILO}>
      33%
    </ProgressBar>
  </Fragment>
</Playground>

### Sizes

You can use the progress bar in a variety of sizes, kilo, mega, or giga.

<Playground>
  <Fragment>
    <ProgressBar max={3} value={1} size={ProgressBar.KILO}>
      1/3
    </ProgressBar>
    <ProgressBar max={3} value={2} size={ProgressBar.MEGA}>
      2/3
    </ProgressBar>
    <ProgressBar max={3} value={3} size={ProgressBar.GIGA}>
      3/3
    </ProgressBar>
  </Fragment>
</Playground>
