# List

- At its core, the `List` component allows you to easily display a related
  content grouped together and organized vertically.
- Generally, every `List` should be used to present simple pieces of
  information. For more complex sets of data, consider using a Data Table.
- A `List` can be Ordered and Unordered. See Variants.

<Playground wrapper={Wrapper}>
  <List ordered size={List.MEGA}>
    <li> A list</li>
    <li> A list</li>
    <li> A list</li>

## Variants

### Ordered

To present content of equal status or value.

<Playground wrapper={Wrapper}>
  <List ordered>
    <li> A list</li>
    <li> A list</li>
    <li> A list</li>
  </List>
</Playground>

### Unordered

Imply sequence and order, and are commonly used when giving a set of instructions.

<Playground wrapper={Wrapper}>
  <List unordered>
    <li> A list</li>
    <li> A list</li>
    <li> A list</li>
  </List>
</Playground>

### Size

The size of the list should be the same as the surrounding content.

## When to use it

- You want to display content related to a single subject. 
- You need to display a big set of data use "Load More" list. This invites the
  user to browse more, shorter load times, combines the advantages of infinite
  scrolling and pagination.
