# Theme reference

In order to define a new theme, you should make a copy of the
[default theme](https://github.com/sumup-oss/circuit-ui/blob/master/src/themes/circuit.js).

Wondering how to make the theme accessible to your components? Read
[Getting started for Developers](Introduction|Developers).

## Colors

`theme.colors.<COLOR_NAME>`

Refer to the [Colors](Styles|Colors) documentation.

## Spacings

`theme.spacings.<SIZE_NAME>`

Refer to the [Spacing](Styles|Spacing) documentation.

## Icon sizes

`theme.iconSizes.<SIZE_NAME>`

Our icon sizes come in `byte`, `kilo`, `mega`, and `giga`.

## Border radius

`theme.borderRadius.<SIZE_NAME>`

## Typography

`theme.typography.<TYPE_ELEMENT>.<SIZE_NAME>.[fontSize|lineHeight]`

Refer to the [Typography](Styles|Typography) documentation.

## Font stack

`theme.fontStack`

Refer to the [Fonts](Styles|Fonts) documentation.

## Font weight

`theme.fontWeight`

Refer to the [Typography](Styles|Typography) documentation.

## Grid

`theme.grid.<BREAKPOINT>.[priority|breakpoint|cols|maxWidth|gutter]`

Refer to the [Grid](Styles|Grid) documentation.

## Breakpoints and media queries

`theme.breakpoints.<BREAKPOINT>`

`theme.mq.<BREAKPOINT>`

Refer to the [Breakpoints and media queries](Styles|Media-Queries) documentation.

## Transitions

`theme.transitions.[default]`

For now there is just one transition. In the future when we
have animations, there may be more :)

## Z-index

`theme.zIndex.[default|absolute|popover|select|tooltip|modal]`

Z-indices for various components to ensure they stack properly.
