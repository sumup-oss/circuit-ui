# Icons

The icons used throughout Circuit UI come from [SumUp's icon library](https://github.com/sumup-oss/circuit-ui/tree/main/packages/icons). The [`@sumup/icons`](https://www.npmjs.com/package/@sumup/icons) package is a required peer dependency of [`@sumup/circuit-ui`](https://www.npmjs.com/package/@sumup/circuit-ui). You can install the package by running the following command in your project:

```sh
# With yarn:
yarn add @sumup/icons
# With npm:
npm install @sumup/icons
```
