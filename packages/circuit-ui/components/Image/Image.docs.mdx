## When to use it

Use the image component when you need to showcase visual files and ensure responsiveness.

## Usage guidelines

- **Do** always specify the `alt` attribute.
- **Do not** use unauthorized images from external sources

## "Alt" text guidelines

- **Do** always write clear and simple `alt` text
- **Do** keep the `alt` text as short as possible
- **Do** write in the active voice
- **Do not** use complicated or uncommon words
