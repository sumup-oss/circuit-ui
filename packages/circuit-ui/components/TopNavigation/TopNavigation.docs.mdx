# TopNavigation

<Status.Stable />

The top navigation is part of the [application shell](https://developers.google.com/web/fundamentals/architecture/app-shell). It contains the branding, page links, and the user profile menu.

## Usage with the side navigation

<Story id="navigation-topnavigation--with-side-navigation" />

When used alongside the [SideNavigation](Navigation/SideNavigation) component, the top navigation displays a hamburger button on narrow viewports which can be used to toggle the side navigation.
